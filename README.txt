Personal Journal - Flutter
-------------------------
Isi project ini adalah kode sumber aplikasi Personal Journal yang terdiri dari beberapa file:
- pubspec.yaml
- lib/main.dart
- lib/models/note.dart
- lib/services/storage_service.dart
- lib/pages/home_page.dart
- lib/pages/add_note_page.dart
- lib/pages/gallery_page.dart

Instruksi singkat:
1. Salin folder 'personal_journal' ke komputer Anda.
2. Jalankan 'flutter pub get' di folder project.
3. Jalankan aplikasi dengan 'flutter run' atau build APK.
